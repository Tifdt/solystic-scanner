# Solystic Scanner — FALORIA & Co

Outil de diagnostic IA pour machines de tri postal Solystic (PIC La Poste Saint-Priest).

## Déploiement Vercel

1. Push ce repo sur GitHub
2. Importer le projet sur vercel.com
3. Dans Settings > Environment Variables : ajouter `VITE_ANTHROPIC_KEY` avec ta clé Anthropic

## Dev local

```bash
npm install
npm run dev
```
